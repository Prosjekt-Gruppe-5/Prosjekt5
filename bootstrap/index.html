Informasjon om oppgaven

    Nettside
    Denne nettsiden er lagt opp slik at du kan registrere deg som student eller foreleser. Her kan studenter velge studieretningen deres og forelesere kan velge faget de underviser. Både student og foreleser kan bytte passord hvis de vil det, det er en liten "bug" hvor passordet blir hashet når man prøver å bytte det. Login-systemet har ingen funksjon for å dehashe det etterpå så dette blir nok ikke fikset i steg 1.

    En student kan sende melding til en foreleser anonymt, ved å velge emne foreleseren underviser i. Foreleseren kan gi svar på meldinger som blir sendt av studenter. For å sende melding som student/foreleser så man legge til /sites/melding_student eller _foreleser i URLen.



    API og app
    Vi har laget et basic api, og en veldig basic app(som ikke er fungerende enda). APIet vårt fungerer ved http requests (Get, post). http://158.39.188.205/steg1/api/api.php
